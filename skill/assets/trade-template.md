# Quick Shorthand Format

For fast requests, use this condensed format:

```
NY AM | 10:15 ET | Price: 21450
PDH: 21500 | PDL: 21350
Bias: Bullish (HH/HL on 4H)
Swept: PDL at 09:35
FVG: 21380-21400 (bullish)
News: None
P&L: $0 | Trades: 0
Setup: Swept PDL, displacement up, FVG left behind
```
